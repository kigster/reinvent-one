# ReinventONE 2.0

This site is made using Claude AI. It runs on ReactJS/Next and TailwindCSS.

## Running It

This site uses ReactJS with NextJS that compile the site into a 100% static site.

To deploy:

```bash
npm install
npm run dev
```

This should start the local server which you can access at [https://localhost:3000/](https://localhost:3000/).
